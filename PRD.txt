AE 算法流程
- 亮度获取：图像经过 OB、shading 后，将图像分成 64*48 的网格，获取亮度信息
- 计算 Target：用 luma target、直方图、bright region、dark region、extreme color 等 AE 信息统计模块分析，统计出最终目标亮度（Target）。基础亮度信息和目标亮度进行比较分析，决定是否收敛
- 曝光调节：若决定收敛，则经过曝光表进入 exp index，将曝光时间和 sensor gain 配置给 sensor，之后经过 gamma、TMC、LTM、GTM 做 ISP gain 补偿，最后获得正确曝光；如果不收敛，则循环到 next AE index，继续获取亮度，直到完成正确曝光

Safe target 怎么得来的
- 计算出 Bright_Low_target、Bright_High_Target、Dark_Low_target 和 Dark_High_Target 四个值，去掉最大值和最小值，得到 low target 和 high target
- 将 reference target luma 与 low target 和 high target 比较，取中间值作为 safe target
- 将 safe target 和 MinTargetAdjRatio*reference_target_luma 和 MaxTargetRatio*reference_target_luma 比较，取中间值为最终 safe target

LTM 算法流程
- 应用双边滤波器以生成非线性 mask
- 避免边缘出现光晕瑕疵
- 使用 LTM 曲线集进行局部增益调整
- 使用 LTM 和 LCE 的组合计算 local gain

TMC 的作用
- 作为 LTM 和 GTM 的混合比例统计。LTM 和 GTM 的占比相加必须等于 1

Lux index 怎么计算
- Lux 指数用于指示当帧亮度超过最大值 exp_index 时测试条件的亮度信息
- 如果帧亮度 = 测试条件的亮度目标，则 lux_index 等于 exp_index

Darkboost gain 和 ADRC gain 怎么来的
- Darkboost gain = long target / safe target
  （取 MaxDarkBoostGain 和 Dark_low_target/safe target 的最小值，再将这个最小值与 MaxADRCGain/DRCGain 的值对比，得到的最小值）
- ADRC gain = safe target / short target
- short target = min(bright_low_target, bright_high_target, safe target)
- ADRC gain = min(safe target / short target, MaxDRCgain)

影响对比度的模块有哪些
- Gamma、LTM、GTM、TMC

Gamma 调试优先还是 LTM 调试优先？
- Gamma（gamma 影响全局亮度，LTM 影响局部亮度，先调好全局再调局部）

帧率是 240 的曝光时间
- Time = 1/帧率 = 1/240 (s) ≈ 4.17 ms

7250 平台 TMC 用的是 3.0 还是 4.0？
（开放问题，需查具体平台 datasheet）

高通 AE 分为多少个数据块统计
- 64*48 个统计数据块，通过中心测光模式统计

高通曝光表怎么调试，主要影响什么
- 首先计算出垂直消隐时间和首行曝光时间
- FrameLengthLines = Frame_length × Line_length（一帧的像素数）
- Vertical Blanking Time = 1 / 最大帧率 / FrameLengthLines × vertical_offset × 10^9
- 首行曝光时间 = 1 / 最大帧率 / FrameLengthLines × minlinecount × 10^9
- 其他行优先以曝光时间形式曝光，为了避免暗环境拖影，最后一行以 gain 优先形式曝光
- 主要影响：AEC 曝光表通过传感器曝光时间和模拟增益控制图像亮度，主要影响亮度和噪声

曝光方程
- AV + TV = BV + SV = EV
- AV + TV：曝光的供给面，有多少光进入摄像头
- BV + SV：曝光的接受面，有多少光反映到图片亮度上
- EV：曝光量，光圈大小和快门时间组合共同决定曝光量

闪光灯打闪过曝
- 调整主闪和预闪的比率 flux，flux 越大闪光灯拍的照片越暗

Video 拍摄到人脸画面偏亮、不稳定
- 调整 video 的收敛速度，使其变慢
- 关闭 face AE，检测到人脸时不调用 face AE 参数，画面无变化，通过 tone mapping 调整人脸

Metering 分为哪两个模块？
- ev table 和 AE 数据统计
- ev table 是各帧的合成调整，用于算法处理
- AE 数据统计就是正在调整的那些

TMC 主要调哪些参数
- TMC 很少调，建议调的参数：GTM 和 LTM 的混合比率、dark boost ratio、tone target 和 tone anchors
